how to run this app..
1. clone using https mode (git clone https://github.com/moronkids/cra-nft.git)
2. open the root folder of downloaded source code (cd cra-nft)
3. type : npm install
4. type : npm start

<br/>
---------------- note --------------<br/>
if you wanna run as e production build<br/>
1. npm run build<br/>
2. npm run serve/ npm serve
